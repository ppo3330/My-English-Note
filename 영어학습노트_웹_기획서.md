# My English Note 웹 기획서

## 1. 기획서 목적

이 문서는 “무엇을 만들 것인지”를 정리한 문서다.

My English Note는 영어 단어, 구동사, 문장을 따로 저장하고 복습할 수 있는 개인용 영어 학습 웹사이트다.

---

## 2. 만들고 싶은 서비스

### 한 줄 설명

영어 공부 중 만난 단어, 구동사, 문장을 기록하고 카드 학습과 직접 입력 시험으로 복습하는 웹사이트.

### 사용 목적

- 단어장을 따로 들고 다니지 않고 웹에 기록한다.
- 단어, 구동사, 문장을 각각 다른 노트처럼 관리한다.
- 저장한 내용을 나중에 카드처럼 보거나 직접 입력하면서 외운다.

---

## 3. 사용자

주 사용자는 영어를 공부하는 개인 사용자 1명이다.

처음 버전에서는 회원가입, 로그인, 다른 사용자 공유 기능은 만들지 않는다.

---

## 4. 노트 구성

웹사이트에는 세 가지 노트가 있다.

### 단어 노트

책이나 문제집에서 외우고 싶은 단어를 저장한다.

필수 입력:

- 영어 단어
- 한국어 뜻

선택 입력:

- 예문
- 메모

### 구동사 노트

회화에 자주 쓰이는 구동사를 저장한다.

필수 입력:

- 구동사
- 한국어 뜻

선택 입력:

- 예문
- 메모

### 문장 노트

영상을 보다가 마음에 드는 문장을 저장한다.

필수 입력:

- 영어 문장
- 한국어 해석

선택 입력:

- 출처
- 메모

---

## 5. 핵심 기능

2주 안에 만들 첫 버전에는 다음 기능만 포함한다.

### 1) 항목 저장

- 노트를 선택한다.
- 영어와 뜻을 입력한다.
- 저장 버튼을 누르면 목록에 추가된다.

### 2) 항목 관리

- 저장한 항목을 볼 수 있다.
- 항목을 수정할 수 있다.
- 항목을 삭제할 수 있다.
- 영어 또는 한국어로 검색할 수 있다.

### 3) 카드 학습

- 영어만 보고 뜻을 떠올린다.
- 뜻만 보고 영어를 떠올린다.
- 카드를 누르면 정답이 보인다.
- 알고 있음, 헷갈림, 모름 중 하나로 표시한다.

### 4) 직접 입력 시험

- 저장한 항목 중 일부가 랜덤으로 나온다.
- 문제를 보고 답을 직접 입력한다.
- 영어 답안은 자동으로 맞는지 확인한다.
- 한국어 뜻이나 문장 해석은 사용자가 직접 맞았는지 판단한다.

### 5) 데이터 저장

- 브라우저 `localStorage`에 데이터를 저장한다.
- 새로고침해도 데이터가 유지된다.
- JSON 파일로 백업하고 복원할 수 있다.

## 7. 화면 구성

### 홈 화면

- 서비스 이름
- 세 노트 선택 버튼
- 최근 저장한 항목 간단 표시

### 노트 화면

- 현재 노트 이름
- 항목 입력 폼
- 저장된 항목 목록
- 검색창
- 카드 학습 시작 버튼
- 시험 시작 버튼

### 카드 학습 화면

- 현재 문제
- 정답 보기 버튼 또는 카드 터치
- 알고 있음 / 헷갈림 / 모름 버튼
- 다음 카드 버튼

### 시험 화면

- 문제
- 답 입력창
- 정답 확인 버튼
- 다음 문제 버튼
- 시험 결과 화면

## 8. 데이터 구조

학습 항목은 다음 형태로 저장한다.

```javascript
{
  id: "고유 ID",
  type: "word", // word | phrasalVerb | sentence
  english: "take off",
  korean: "벗다, 이륙하다",
  example: "The plane took off.",
  memo: "",
  source: "",
  status: "new", // new | known | unsure | unknown
  createdAt: "생성 날짜",
  updatedAt: "수정 날짜"
}
```

## 9. 완료 기준

첫 버전은 다음이 가능하면 완료로 본다.

- 세 노트를 따로 사용할 수 있다.
- 항목을 추가, 수정, 삭제할 수 있다.
- 새로고침해도 데이터가 남아 있다.
- 카드 학습을 할 수 있다.
- 직접 입력 시험을 할 수 있다.
- JSON 백업과 복원이 가능하다.
- 모바일과 PC에서 사용할 수 있다.
